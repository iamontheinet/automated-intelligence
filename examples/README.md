# Examples

This directory contains example notebooks and SQL scripts demonstrating various Snowflake AI and data quality features.

## Files

- `ai_functions_examples.sql` - SQL examples of Cortex AI functions
- `ai_functions_notebook.ipynb` - Jupyter notebook demonstrating AI functions
- `Dash_AI_DT.ipynb` - Notebook exploring Dynamic Tables and AI features

## Purpose

These files are reference examples and teaching materials, not part of the core setup or demo execution.

## Usage

Explore these files to learn about Snowflake AI capabilities and implementation patterns used throughout the project.
